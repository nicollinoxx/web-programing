Understanding White Space

In HTML, white space is managed using line breaks and paragraph tags. The <p> tag defines a paragraph, automatically adding space before and after the content.

For example: <p>This is a paragraph.</p> creates a block of text with margins around it.

If you want to break a line within the same paragraph without creating a new block, use the <br> tag. For example:

This is the first line.
This is the second line.

This will display the two sentences on separate lines with no extra spacing between them.
