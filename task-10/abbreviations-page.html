Common Abbreviations

The abbreviation HTML stands for HyperText Markup Language, which is the standard language for creating web pages.

Here are some more common abbreviations:

  • CSS - Used for describing the presentation of a document written in HTML or XML.
  • JS - A programming language that enables interactive web pages.
  • API - A set of functions and procedures allowing the creation of applications that access the features or data of an operating system, application, or other services.
